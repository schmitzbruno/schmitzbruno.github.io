🧩 Quadro de Rotinas para Crianças com Autismo.  

📜 Projeto desenvolvido para auxiliar educadores na organização de atividades lúdicas para crianças com espectro autista, de forma visual, interativa e acessível.  

👉 https://schmitzbruno.github.io/

✨ Funcionalidade
- Interface intuitiva: Crianças interagem clicando em imagens ilustrativas.  
- Feedback auditivo: Sons explicativos (ex: *"Hora da higiene"*) ao clicar em determinada atividade.  
- Responsividade: Adapta-se a celulares, tablets e notebooks.  
- Zero burocracia: Não requer login, instalação ou configuração.  

🛠️ Tecnologias Utilizadas  
- Frontend: HTML5, CSS3, JavaScript.  
- Acessibilidade: Foco em design inclusivo e interação simples.  


📱 Como Usa  
1. Acesse o link do projeto em qualquer dispositivo.  
2. A criança ou a professora clica na imagem da atividade desejada (ex: lavar as mãos, brincar, lanchar).  
3. O sistema reproduz o áudio correspondente à atividade.  


🌟 Por Que Este Projeto?  
- Inclusivo: Projetado especificamente para o espectro autista.  
- Universal: Funciona em qualquer dispositivo com navegador.  
- Educativo: Estimula a autonomia e a compreensão de rotinas através da interatividade, imagens e sons.
